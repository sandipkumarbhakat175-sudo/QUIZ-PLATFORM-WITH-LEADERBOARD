# QUIZ-PLATFORM-WITH-LEADERBOARD
🏆 Interactive Quiz Platform with LeaderboardA lightweight, responsive, and fully interactive Quiz Platform built entirely with frontend technologies: HTML5, CSS3, and Vanilla JavaScript. 
1. This application challenges users with multiple-choice questions, tracks their scores in real-time, and logs high scores to a dynamic leaderboard.
2. ✨ FeaturesDynamic Quiz Interface: Renders multiple-choice questions dynamically with smooth state transitions.
3. Persistent Leaderboard: Saves player names and high scores locally so rankings persist even after refreshing the page.
4. Instant Score Feedback: Highlights correct and incorrect options immediately upon submission.
5. Countdown Timer: Adds an extra challenge by limiting the completion time per question or session.
6. Fully Responsive Design: Styled from scratch to provide a seamless user experience on mobile, tablet, and desktop screens.
7. No Framework Dependencies: Built purely with standard web APIs to maximize performance and learning depth.
8. 🛠️ Tech StackHTML5: Structured semantic layout for the quiz dashboard, questions container, and leaderboard panel
9. .CSS3: Custom modern styling utilizing Flexbox, custom variables, hover micro-interactions, and animations.
10. JavaScript (ES6+): Handles game initialization logic, core DOM manipulation, countdown event handling, and score calculations.
11. question.js: it will  be the contained quiz question form a java script file.
